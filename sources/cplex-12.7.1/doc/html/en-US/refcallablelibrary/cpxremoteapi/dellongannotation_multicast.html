CPXXdellongannotation_multicast and CPXdellongannotation_multicast

The routine CPXXdellongannotation_multicast/CPXdellongannotation_multicast deletes a long valued annotation.

int CPXXdellongannotation_multicast( CPXENVGROUPptr group, CPXDIM idx )

int CPXdellongannotation_multicast( CPXENVGROUPptr group, int idx )

Description

Multicast version

Arguments

Name Description
group
idx An integer that specifies the numeric index of the last annotation tag to be deleted.

Return

The routine returns 0 (zero) if successful and nonzero if an error occurs.

Example

status = CPXdellongannotation_multicast (env, lp, 0);

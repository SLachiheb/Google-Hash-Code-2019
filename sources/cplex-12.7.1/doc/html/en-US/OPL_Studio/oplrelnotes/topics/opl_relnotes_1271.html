Release notes for IDE and OPL V12.7.1

These release notes highlight changes in the IDE and OPL V12.7.1. Please review these notes before using IDE and OPL V12.7.1.

Here are the release notes for IDE and OPL V12.7.1

Support of CPLEX annotations for Benders decomposition in OPL script

In 12.7.1, support for the CPLEX annotations API and Benders strategy settings are available in OPL. You can now annotate your model to guide CPLEX in a Benders decomposition, using names of the annotations and a constraint or a variable or an array of constraints or an array of variables.

Here is the list of methods:
  • newNumAnnotation
  • newLongAnnotation
  • setNumAnnotation
  • setNumAnnotations
  • setLongAnnotation
  • setLongAnnotations
  • delNumAnnotation
  • delLongAnnotation
  • getNumAnnotation
  • getLongAnnotation
  • getLongAnnotations
  • getNumAnnotations
  • hasNumAnnotation
  • hasLongAnnotation
  • numNumAnnotations
  • numLongAnnotations
  • readAnnotations
  • writeAnnotations
  • writeBendersAnnotation
Benders strategy can be set to a CPLEX object using a JavaScript call:
<code>
execute{
    cplex.bendersstrategy = 3;
}
</code>

A full example is provided under CPLEXSTUDIO_DIR127/opl/examples/opl/benders and can be imported in the IDE. This example provides an example of how to use the annotations system to annotate variables and constraints, either 1 by 1, or via arrays.

Release notes for IDE and OPL V12.7.0

These release notes highlight changes in the IDE and OPL V12.7.0. Please review these notes before using IDE and OPL V12.7.0.

Here are the release notes for IDE and OPL V12.7.0

SPSS Connector

SPSS Connector and support in the IDE have been removed.

DBLink

DBLink has been removed. For an alternative way to connect data see Connecting to data in CPLEX Optimization Studio V12.7 .

CPLEX Server connection

Connection from the IDE to CPLEX Server is no longer possible. Use DOcplexcloud instead. For more information see Solving OPL models on the cloud.

These notes announce elements deprecated in or removed from CPLEX V12.6.3.

Deprecated convert utility

Previous versions of CPLEX distributed convert, a utility to convert fixed-format MPS, BAS, or ORD files into a format that CPLEX supports even when the file lacked names of bounds or righthand sides, or when the file contained blank space within a name, blank lines, or uncommented characters.

The convert utility has been deprecated. It will be removed in a future release.

Removed memory managers from Callable Library (C API)

The following routines of the Callable Library (C API) were deprecated in CPLEX V12.4. They have been removed from CPLEX V12.6.3.

  • CPXmalloc
  • CPXrealloc
  • CPXfree

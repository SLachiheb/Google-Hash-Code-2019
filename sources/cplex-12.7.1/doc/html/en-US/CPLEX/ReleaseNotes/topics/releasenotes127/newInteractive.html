New Interactive Optimizer features in CPLEX V12.7.0

Announces new features of CPLEX V12.7.0 in the Interactive Optimizer.

These notes highlight new features in the Interactive Optimizer of interest to users of CPLEX V12.7.0.

Evaluating variability

CPLEX V12.7.0 offers a new command in the Interactive Optimizer to help users evaluate problem variability in their mixed integer programs, MILP, MIQP, and MIQCP. See the new topic Evaluating variability in the reference manual of the Interactive Optimizer for more detail.

Piecewise linear (PWL) constraints

The Interactive Optimizer now supports piecewise linear (PWL) constraints. For more detail about commands and syntax for PWL in your model, see Piecewise linear (PWL) in the Interactive Optimizer in the reference manual of the Interactive Optimizer.

Annotating a model

CPLEX V12.7.0 offers support so that you can explicitly annotate objects in your model. In particular, the Interactive Optimizer accepts an annotation file with the file extension .ann for that purpose. For more detail about annotating a model, see also Annotating a model in CPLEX V12.7.0 in these release notes.

Benders algorithm

A new parameter Benders strategy makes it possible to apply Benders algorithm to solve a linear program (LP) or mixed integer linear program (MILP) in the Interactive Optimizer. To do so, follow these steps:
  1. Read in your annotated model from a formatted .ann file.
  2. By means of the Benders strategy parameter, specify the Benders strategy that you prefer. For example, 
    set benders strategy 2
  3. Invoke the optimization command.

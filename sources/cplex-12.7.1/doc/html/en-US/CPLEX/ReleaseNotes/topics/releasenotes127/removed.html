Removed

Documents elements removed from CPLEX V12.7.0.

These elements were deprecated in earlier versions of CPLEX and removed from CPLEX V12.7.0.

Error codes removed

These error codes have been removed from CPLEX 12.7.0.

  • 1216 CPXERR_NO_RNGVAL
  • 3603 CPXERR_PTHREAD_CREATE
  • 3601 CPXERR_PTHREAD_MUTEX_INIT

Parameters removed

This parameter was deprecated in a previous version of CPLEX: CPX_PARAM_SOLUTIONTARGET, SolutionTarget, solutiontarget; these symbols have been removed from CPLEX V12.7.0. Use optimality target instead.

  • In the Callable Library (C API), these constants have been removed:
    • CPX_SOLUTIONTARGET_AUTO
    • CPX_SOLUTIONTARGET_OPTIMALCONVEX
    • CPX_SOLUTIONTARGET_FIRSTORDER
    • CPX_SOLUTIONTARGET_OPTIMALGLOBAL
  • In the C++ API, the field IloCplex::IntParam::SolutionTarget and the enumeration IloCplex::SolutionTargetType have been removed.
  • In the Java API, IloCplex.IntParam.SolutionTarget and the inner class IloCplex.SolutionTarget have been removed.
  • In the .NET API, Cplex.IntParam.SolutionTarget and Cplex.SolutionTarget have been removed.
  • In the Python API, the constants of solution_target_constants have been removed.

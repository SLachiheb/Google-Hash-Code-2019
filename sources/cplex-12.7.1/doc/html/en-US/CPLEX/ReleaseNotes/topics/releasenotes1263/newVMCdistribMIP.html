New features for users of VMC files for distributed MIP optimization in CPLEX V12.6.3

Announces new features of CPLEX V12.6.3 for users of VMC files in distributed MIP optimization.

These notes highlight new features of interest to users of virtual machine configuration (VMC) files for distributed MIP optimization in CPLEX V12.6.3.

CPLEX V12.6.3 offers new features that you can specify in a virtual machine configuration (VMC) file for your applications of distributed MIP optimization. CPLEX now lets you set parameters exclusively for ramp up. You can also tell CPLEX whether to limit the effects of presolve on potentially optimal solutions found during ramp up. You can also now designate some workers as unlikely to win the ramp up phase. For more detail, see the topic VMC file for specifying parameters, ramp up options, and environment variables in distributed parallel optimization in the CPLEX User's Manual.

New .NET features in CPLEX V12.7.0

Announces new features of CPLEX V12.7.0 in the .NET API.

These notes highlight new features in the .NET API of interest to users of CPLEX V12.7.0.

Benders in the .NET API

To use the new Benders features in your .NET application, optionally annotate your model; then set the new parameter Benders strategy accordingly, and invoke optimization with Cplex.Solve.

For more about Benders algorithm in CPLEX V12.7.0, see also the announcement Benders algorithm in CPLEX V12.7.0 in these release notes.

Annotations

In the .NET API, CPLEX V12.7.0 offers new features to support annotation of your model. For more about annotations that support Benders decomposition as a feature of CPLEX V12.7.0, see also the announcement Annotating a model in CPLEX V12.7.0 in these release notes.

CPLEX V12.7.0 offers a new method Cplex.Remove to remove a designated callback from your model.

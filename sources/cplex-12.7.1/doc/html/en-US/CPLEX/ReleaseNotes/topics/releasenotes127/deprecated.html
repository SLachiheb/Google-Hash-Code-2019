Deprecated

Documents elements deprecated in CPLEX V12.7.0.

These notes announce elements deprecated in CPLEX V12.7.0. They will be removed from a future release. In the Interactive Optimizer, new commands serve the purpose of these deprecated commands. Likewise, in the application programming interfaces (APIs), other methods serve the purpose of these deprecated methods; for example, in the Java API, in order to count the number of cuts, use the method getNcuts with an argument to designate the type of cuts to count. Likewise, in the .NET API, use GetNcuts with an appropriate argument.

Interactive Optimizer commands deprecated in CPLEX V12.7.0

The Interactive Optimizer command conflict has been deprecated. Use the new command tools conflict instead. The behavior has not changed; only the syntax of the command has changed.

The Interactive Optimizer command tune has been deprecated. Use the new command tools tune instead. The behavior has not changed; only the syntax of the command has changed.

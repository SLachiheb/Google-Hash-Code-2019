Changes in the Interactive Optimizer

Announces changes introduced by CPLEX V12.7.0.

These notes explain changes introduced by CPLEX V12.7.0 in the Interactive Optimizer.

In addition to the new command, announced by New Interactive Optimizer features in CPLEX V12.7.0 in these release notes, two existing commands, conflict and tune, have changed their syntax.

Conflict refiner

Previously, to invoke the conflict refiner on a model in the Interactive Optimizer, you used the command: 
conflict
Now, you invoke the new tools command with the option conflict, like this: 
tools conflict
The behavior of the conflict refiner remains the same; only the syntax of the invoking command has changed.

Tuning a model or group of models

Previously, to tune a model or groups of models in the Interactive Optimizer, you used the command: 
tune filename parameters.prm
Now, you invoke the new tools command with the option tune, followed by the filename of the model or models to tune, followed by an optional PRM file of parameter settings, like this: 
tools tune filename parameters.prm
The behavior of the tuner remains the same; only the syntax of the invoking command has changed.

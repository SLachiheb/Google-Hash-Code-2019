New Callable Library (C API) features in CPLEX V12.6.3

Announces new features of CPLEX V12.6.3 in the Callable Library (C API).

These notes highlight new features in the Callable Library (C API) of interest to users of CPLEX V12.6.3.

New support for Community Edition in the Callable Library (C API)

To enable users of the Community Edition of IBM ILOG CPLEX Optimization Studio to verify that the size of their model does not exceed limits of the Community Edition, CPLEX V12.6.3 now offers new routines.

  • CPXXgetnumlazyconstraints and CPXgetnumlazyconstraints CPXXgetnumlazyconstraints, CPXgetnumlazyconstraints to count the number of lazy constraints in a model.
  • CPXXgetnumusercuts and CPXgetnumusercuts CPXXgetnumusercuts, CPXgetnumusercuts to count the number of user cuts in a model.

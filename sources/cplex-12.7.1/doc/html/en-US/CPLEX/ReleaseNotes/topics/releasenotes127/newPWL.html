Piecewise linear (PWL) features in CPLEX V12.7.0

Announces new features and extensions of piecewise linearity in CPLEX V12.7.0.

CPLEX V12.7.0 extends support of piecewise linearity through multiple file formats (LP, MPS, SAV, and XML) and across APIs (C, C++, Java, .NET, Python).

LP file format and PWL

LP file format now supports declaration of piecewise linear constraints in a model in a new section titled PwlConstr. For documentation of the syntax, see Piecewise linear (PWL) constraints in LP file format in the reference manual of File formats supported by CPLEX.

MPS file format and PWL

MPS file format now also supports declaration of piecewise linear constraints in a model. For more about declarations of PWL constraints in an MPS file, see the topic Piecewise linear (PWL) constraints in MPS files in the reference manual File formats supported by CPLEX.

SAV file format and PWL

In CPLEX V12.7.0, SAV file format supports piecewise linear constraints in a model. That is, when you write a model that includes piecewise linear constraints in SAV file format (a binary format specific to CPLEX), the piecewise linearity will be preserved.

XML file format and PWL

XML file format to support serialization of a model also supports piecewise linear constraints in a model. That is, when you write a model that includes piecewise linear constraints in XML file format, the piecewise linearity will be preserved. To be sure that CPLEX extracts the piecewise linear constraints as you intend, you must specify the breakpoints in increasing order of their x-coordinates.

PWL and Interactive Optimizer

CPLEX V12.7.0 support piecewise linear constraints in models in the Interactive Optimizer. For more detail, see the topic Piecewise linear (PWL) in the Interactive Optimizer in the Interactive Optimizer Reference Manual.

PWL and Callable Library (C API)

New routines of the Callable Library (C API) extend support of piecewise linearity in CPLEX V12.7.0. The announcement New Callable Library (C API) features in CPLEX V12.7.0 includes links to reference documentation of these new routines.

PWL in the C++ API

CPLEX already supports piecewise linear (PWL) expressions in the C++ API. See, for example, ilotransport.cpp and the topics detailing that example Using piecewise linear functions in optimization: a transport example in the CPLEX User's Manual.

PWL in the Java API

CPLEX already supports piecewise linear (PWL) expressions in the Java API. See, for example, Transport.java and the topics detailing that example Using piecewise linear functions in optimization: a transport example in the CPLEX User's Manual.

PWL in the .NET API

CPLEX already supports piecewise linear (PWL) expressions in the .NET API. See, for example, Transport.cs and the topics detailing that example Using piecewise linear functions in optimization: a transport example in the CPLEX User's Manual.

PWL in the Python API

The announcement New Python features in CPLEX V12.7.0 in these release notes lists the new methods in the Python API to support piecewise linear (PWL) constraints. Also, the product includes a new example, transport.py, using the new methods.

New Python features in CPLEX V12.6.3

Announces new features of CPLEX V12.6.3 in the Python API.

These notes highlight new features in the Python API of interest to users of CPLEX V12.6.3.

New support for with statement in the Python API

In CPLEX V12.6.3, the class Cplex is now a context manager and can be used in a with statement.

New Cplex.end method in Python API

CPLEX V12.6.3 offers a new method Cplex.end in the Python API.

New support for Community Edition in the Python API

To enable users of the Community Edition of IBM ILOG CPLEX Optimization Studio to verify that the size of their model does not exceed limits of the Community Edition, CPLEX V12.6.3 now offers new methods.

  • Cplex.linear_constraints.advanced.get_num_lazy_constraints counts the number of lazy constraints in a model.
  • Cplex.linear_constraints.advanced.get_num_user_cuts counts the number of user cuts in a model.

New parameters in CPLEX V12.7.0

CPLEX V12.7.0 offers new parameters.

Benders strategy

CPLEX V12.7.0 offers a new parameter Benders strategy to specify to CPLEX to apply Benders algorithm as a strategy to solve your model. (See Benders algorithm in CPLEX V12.7.0 in these release notes for more about Benders algorithm as supported in CPLEX V12.7.0.)

Benders feasibility cut tolerance

CPLEX V12.7.0 offers a new parameter Benders feasibility cut tolerance to specify to CPLEX a tolerance for feasibility cuts.

Benders optimality cut tolerance

CPLEX V12.7.0 offers a new parameter Benders optimality cut tolerance to specify to CPLEX a tolerance for optimality cuts.

RLT cuts

CPLEX V12.7.0 offers a new parameter Reformulation Linearization Technique (RLT) cuts that turns on or off the generation of cuts based on the Reformulation Linearization Technique. These cuts, also known as RLT cuts, can improve performance when CPLEX solves certain nonconvex quadratic programs (QP) or mixed integer quadratic programs (MIQP) solved to global optimality. This new parameter interacts with the existing parameter optimality target, documented in the CPLEX Parameters Reference Manual.

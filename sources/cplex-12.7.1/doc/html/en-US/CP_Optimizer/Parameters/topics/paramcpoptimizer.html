Parameters of CP Optimizer

Users can browse CP Optimizer parameters, organized by topics.

In CP Optimizer, the parameters are represented by the enumerations IntParam and NumParam. In the C++ API, these enumerations are IloCP::IntParam and IloCP::NumParam.

Display and output

  • LogVerbosity. Possible values are IloCP::Quiet, IloCP::Terse, IloCP::Normal, and IloCP::Verbose.
  • LogPeriod. Possible values are positive integers.
  • WarningLevel. Possible values are integers from 0 to 3.
  • PrintModelDetailsInMessages. Possible values are IloCP::On and IloCP::Off.
  • ModelAnonymizer. Possible values are IloCP::On and IloCP::Off.
  • UseFileLocations. Possible values are IloCP::On and IloCP::Off.

Presolve

  • Presolve. Possible values are IloCP::On and IloCP::Off.

Optimality tolerances

  • OptimalityTolerance. Possible values are floating-point values from 0.0 to IloInfinity.
  • RelativeOptimalityTolerance. Possible values are floating-point values from 0.0 to IloInfinity.

Search control

  • Workers. Possible values are IloCP::Auto and positive integers.
  • SearchType. Possible values are IloCP::Auto, IloCP::DepthFirst, IloCP::Restart, and IloCP::MultiPoint.
  • RandomSeed. Possible values are integers from 0 to IloIntMax.
  • RestartFailLimit. Possible values are integers from 0 to IloIntMax.
  • RestartGrowthFactor. Possible values are floating-point values from 1.0 to IloInfinity.
  • DynamicProbing. Possible values are IloCP::Auto, IloCP::On, and IloCP::Off.
  • DynamicProbingStrength. Possible values for this parameter are floating-point values from 0.001 to 1000.
  • MultiPointNumberOfSearchPoints. Possible values are integers from 0 to IloIntMax.
  • TemporalRelaxation. Possible values are IloCP::On and IloCP::Off.
  • FailureDirectedSearch. Possible values are IloCP::On and IloCP::Off.
  • FailureDirectedSearchEmphasis. Possible values are IloCP::Auto and floating-point values from 1.0 to IloInfinity.
  • FailureDirectedSearchMaxMemory. Possible values are integers from 0 to IloIntMax.
  • AutomaticReplay. Possible values are IloCP::On and IloCP::Off.

Search limits

  • TimeLimit. Possible values are floating-point values from 0.0 to IloInfinity.
  • TimeMode. Possible values are IloCP::ElapsedTime and IloCP::CPUTime.
  • FailLimit. Possible values are integers from 0 to IloIntMax.
  • ChoicePointLimit. Possible values are integers from 0 to IloIntMax.
  • BranchLimit. Possible values are integers from 0 to IloIntMax.
  • SolutionLimit. Possible values are integers from 0 to IloIntMax.

Inference levels for constraint propagation

  • DefaultInferenceLevel. Possible values are IloCP::Low, IloCP::Basic, IloCP::Medium, and IloCP::Extended.
  • AllDiffInferenceLevel. Possible values are IloCP::Default, IloCP::Low, IloCP::Basic, IloCP::Medium, and IloCP::Extended.
  • DistributeInferenceLevel. Possible values are IloCP::Default, IloCP::Low, IloCP::Basic, IloCP::Medium, and IloCP::Extended.
  • CountInferenceLevel. Possible values are IloCP::Default, IloCP::Low, IloCP::Basic, IloCP::Medium, and IloCP::Extended.
  • CountDifferentInferenceLevel. Possible values are IloCP::Default, IloCP::Low, IloCP::Basic, IloCP::Medium, and IloCP::Extended.
  • SequenceInferenceLevel. Possible values are IloCP::Default, IloCP::Low, IloCP::Basic, IloCP::Medium, and IloCP::Extended.
  • AllMinDistanceInferenceLevel. Possible values are IloCP::Default, IloCP::Low, IloCP::Basic, IloCP::Medium, and IloCP::Extended.
  • ElementInferenceLevel. Possible values are IloCP::Default, IloCP::Low, IloCP::Basic, IloCP::Medium, and IloCP::Extended.
  • PrecedenceInferenceLevel. Possible values are IloCP::Default, IloCP::Low, IloCP::Basic, IloCP::Medium, and IloCP::Extended.
  • IntervalSequenceInferenceLevel. Possible values are IloCP::Default, IloCP::Low, IloCP::Basic, IloCP::Medium, and IloCP::Extended.
  • NoOverlapInferenceLevel. Possible values are IloCP::Default, IloCP::Low, IloCP::Basic, IloCP::Medium, and IloCP::Extended.
  • CumulFunctionInferenceLevel. Possible values are IloCP::Default, IloCP::Low, IloCP::Basic, IloCP::Medium, and IloCP::Extended.
  • StateFunctionInferenceLevel. Possible values are IloCP::Default, IloCP::Low, IloCP::Basic, IloCP::Medium, and IloCP::Extended.

Failure explanation

  • LogSearchTags. Possible values are IloCP::On and IloCP::Off.

Conflict refiner

  • ConflictRefinerTimeLimit. Possible values are floating-point values from 0.0 to IloInfinity.
  • ConflictRefinerIterationLimit. Possible values are integers from 0 to IloIntMax.
  • ConflictRefinerBranchLimit. Possible values are integers from 0 to IloIntMax.
  • ConflictRefinerFailLimit. Possible values are integers from 0 to IloIntMax.
  • ConflictRefinerOnVariables. Possible values are IloCP::On and IloCP::Off.

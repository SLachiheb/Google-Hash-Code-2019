Changed feature: Setting search phases

A new API for setting search phases is available.

The methods for setting search phases were changed for this version. Instead of calling the search with a search phase or phases, the phases are set by using new setter functions:

  void IloCP::setSearchPhases(const IloSearchPhase phase);
  void IloCP::setSearchPhases(const IloSearchPhaseArray phaseArray);

in the C++ API. The functions

  void IloCP::startNewSearch(const IloSearchPhaseArray phaseArray) const;
  void IloCP::startNewSearch(const IloSearchPhase phase) const;
  IloBool IloCP::solve(const IloSearchPhaseArray phaseArray) const;
  IloBool IloCP::solve(const IloSearchPhase phase) const;

are deprecated as of this version. In addition, the function void IloCP::setSearchPhases(); was added for clearing the search phase.

Corresponding functions were added and deprecated in the Java™ API and C# API.

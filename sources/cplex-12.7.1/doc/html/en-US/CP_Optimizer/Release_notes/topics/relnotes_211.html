ILOG CP Optimizer 2.1.1 Release Notes

Presents the known issues resolved in ILOG CP Optimizer 2.1.1.

The following issues have been resolved since ILOG CP Optimizer 2.1:

Resolved issues Details
CP-236 When CP Optimizer is used from OPL, pow(x,y) may force x to be positive even when y is integral.
CP-237 Crash with IloStep and CumulFunctionInferenceLevel=Extended.
CP-241 Java and C# accept runtime licenses but throw a bogus exception.
CP-246 Search incompleteness due to startAfterStart precedence.
CP-253 IloForbidStart/End/Extent are not imposed if used more than 32 times on the same interval variable.
OPL-4281 When CP Optimizer is used from OPL, there is an unexpected error when a stepFunction array is indexed in a forall.

The following product enhancements have been made since ILOG CP Optimizer 2.1:

Enhancements Details
CP-247 Improved performance of propagation at the root node when using IloPack together with very large domains.
CP-252 Improved propagation when interval variables are involved in multiple IloSpan constraints.

Changes since CP Optimizer V12.6.3

Describes the changes for the release of the CP Optimizer feature of CPLEX® Optimization Studio V12.7.0.

  • Changed feature: Setting search phases
    A new API for setting search phases is available.
  • Changed feature: IlcIntMax and IlcIntMin
    The constants IlcIntMax and IlcIntMin changed and new constants to represent the smallest and largest integers were introduced.
  • New feature: Starting points in the CPO File Format
    Starting points can now be represented in the CPO File Format.
  • New feature: Interactive optimizer
    An interactive optimizer is now available.
  • Performance improvement
    This version has significant performance improvement for scheduling problems.
  • Resolved Issues
    These are the registered issues that are fixed in the CP Optimizer feature of CPLEX Optimization Studio V12.7.0.

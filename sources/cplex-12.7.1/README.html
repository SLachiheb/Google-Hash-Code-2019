IBM ILOG CPLEX Optimization Studio V12.7.1

IBM® ILOG® CPLEX® Optimization Studio est le moyen le plus rapide de créer des modèles d'optimisation efficaces et des applications de pointe permettant de traiter les multiples problèmes de planification et d'ordonnancement. Avec son environnement de développement intégré, son langage de modélisation descriptive, ses outils intégrés et ses solveurs avancés, traitant les problèmes de programmation mathématique et les problèmes de programmation par contraintes, il prend en charge le processus complet de développement et de résolution des modèles.

CPLEX Optimization Studio comprend plusieurs composants. Certains composants ne sont pas disponibles sur toutes les plateformes. Pour en savoir plus, rendez-vous sur la page relative à la configuration système requise qui est décrite en détail ci-dessous. Chacun des composants est placé dans un sous-dossier (ou sous-répertoire pour AIX® et Linux) de l'emplacement d'installation. Les API et les connecteurs pour chaque composant se trouvent dans le sous-dossier (sous-répertoire) de ce composant. Par exemple, les connecteurs MATLAB et AMPL pour CPLEX se trouvent dans le sous-dossier (sous-répertoire) cplex.

  • opl : environnement de modélisation OPL et environnement de développement intégré (IDE)
  • cplex : optimiseurs CPLEX incluant les optimiseurs de simplexe, de barrière et mixte en nombres entiers
  • cpoptimizer : optimiseur PPC (CP Optimizer), qui utilise la technologie de programmation par contraintes
  • concert : couche de modélisation C++ à utiliser avec les optimiseurs

Autres répertoires :

  • doc : version HTML de la documentation, ainsi que des tutoriels au format PDF
  • jre : environnement d'exécution Java™ utilisé pour les processus d'installation et de désinstallation (sur les plateformes dépourvues d'IDE OPL)
  • license : documents de contrat de licence
  • properties : informations d'inventaire
  • Uninstall : informations nécessaires pour la désinstallation du produit

Installation sous Windows

Sous Windows, l'emplacement d'installation par défaut d'IBM ILOG CPLEX Optimization Studio est :
C:\Program Files\IBM\ILOG\CPLEX_Studio1271

L'emplacement d'installation de l'édition Community est :
C:\Program Files\IBM\ILOG\CPLEX_Studio_Community1271

Lors de l'installation, vous avez peut-être sélectionné un emplacement différent ([rép_install]).

IDE

Vous pouvez lancer l'IDE en utilisant une entrée du menu Démarrer :

Démarrer > Tous les programmes > IBM ILOG > CPLEX Optimization Studio (64 bits) 12.7.1 > CPLEX Studio IDE dans le cas d'une installation Windows 64 bits.

Pour l'édition Community :

Démarrer > Tous les programmes > IBM ILOG > ILOG CPLEX Optimization Studio Community Edition (64 bits) 12.7.1 > CPLEX Studio IDE dans le cas d'une installation Windows 64 bits.

Installation sous AIX, Linux et OS X

Les programmes d'installation pour AIX, Linux et OS X sont des fichiers exécutables. Pour lancer une installation, vous devez leur donner un droit d'exécution et les exécuter depuis une ligne de commande. Par exemple, lancez la commande ./<nom_installateur>.bin où <nom_installateur> est le nom du programme d'installation. Vous pouvez donner l'autorisation d'exécution à un fichier avec la commande chmod u+x <nom_installateur>.bin.

Sous AIX, l'emplacement d'installation par défaut d'IBM ILOG CPLEX Optimization Studio est :
/opt/IBM/ILOG/CPLEX_Studio[édition]1271

Sous Linux, l'emplacement d'installation par défaut d'IBM ILOG CPLEX Optimization Studio est :
/opt/ibm/ILOG/CPLEX_Studio[édition]1271

Sous OS X, l'emplacement d'installation par défaut d'IBM ILOG CPLEX Optimization Studio est :
/Users/[utilisateur]/Applications/IBM/ILOG/CPLEX_Studio[édition]1271

où [édition] peut être :
* Community, pour l'édition Community
* vide, pour l'édition commerciale standard

Lors de l'installation, vous avez peut-être sélectionné un emplacement différent ([rép_install]).

IDE

Vous pouvez lancer l'IDE depuis un terminal Linux.
1. Ouvrez une fenêtre de terminal
2. Passez dans le répertoire [rép_install]/opl/oplide
3. Entrez la commande ./oplide

Vous pouvez également entrer le chemin d'accès absolu à partir de n'importe quel emplacement du terminal :
[rép_install]/opl/oplide/oplide
